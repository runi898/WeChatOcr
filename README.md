# 微信 OCR · 截图识别 & 翻译工具 (独立单文件版)

它是一款极其轻量化的 Windows 截图文字识别及翻译小工具，通过调用你电脑上的绿屏提取工具（如微信内置的 OCR 引擎）和各种国内外的在线翻译通道来呈现精准识别！

## 🚀 快速使用

- **即开即用**：所有依赖早已内嵌进了为您打包的单文件版本中。不要安装任何多余代码库或插件，无需保留其它文件！只需双击 `dist/WeChatOCR_Tool_v5.exe` 即可运行（任务栏托盘会有图标运行提示）。
- **默认全局热键**（可在设置面板中更改）：
  - `Alt + 1` ：截图并弹出翻译
  - `Alt + 2` ：仅截图并提取画面文字，静默复制到系统剪贴板

## ⚙️ 翻译引擎介绍与配置

本程序内置了：**腾讯机器翻译（默认优先）**、**百度翻译**、**有道翻译**、**MyMemory** 等通道机制。如果主要翻译服务失败，它会自动走降级通道备用呈现！

### 专属设置：申请官方免费“腾讯机器翻译API”
为保障长效稳定、无缝的精准翻译体验，强烈建议使用个人的 `腾讯云机器翻译 API (TMT)`。腾讯云每个月会对普通用户免费发放 500万 字符（约合几本小说的字数），日常完全足够！

**如何获取：**
1. 访问 腾讯云 API 密钥控制台： [https://console.cloud.tencent.com/cam/capi](https://console.cloud.tencent.com/cam/capi)
2. 登录账号后，点击“**新建密钥**”。
3. 复制生成的 **`SecretId`** 和 **`SecretKey`**。
4. 在本项目运行界面的设置面板下，选择**腾讯翻译**引擎，并将复制的信息填入弹出框。这配置会永久自动保存在你的私人系统数据中（`%APPDATA%/wechatocr`）。

### 软件开机启动的方法
win7和win8系统：
在系统的开始菜单里，有个“启动”文件夹，打开启动文件夹后，只需把要开机自启动的程序的快捷方式放入该文件夹，系统登录成功后就会自动打开设置的软件。
win10系统：
把需要自启动的应用程序的快捷方式放到“系统启动文件夹”里，系统启动文件夹路径如下：
C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp


## 🔧 疑难解答

- **双击没反应 / “无内容可翻译”报错？**
  - 这个打包好的单文件通过解压自身携带的 DLL 与文件来实现，运行它至少确保你系统中曾经**安装并正常运行过 PC 版微信**（需要微信本身的辅助基础库支持）。
- **如何退出？**
  - 此程序采用静默守护，您可以随时到右下角“系统托盘区”找到有“wxocr”字样的图标，右键点击“退出”即可。

## 📦 二维码功能底层依赖

本项目中的二维码功能分别调用了以下开源库和模块：

1. **二维码查知识别 (扫码)**：
   - 依赖于 **`opencv-contrib-python`** 库。
   - 底层调用了开源的微信二维码引擎：`cv2.wechat_qrcode_WeChatQRCode()`。这是 Tencent 官方开源并在 OpenCV Contrib 中提供的基于 CNN（卷积神经网络）的超强解析模型，即使残缺、模糊或极端视角依然能精准识别。
2. **二维码生成**：
   - 依赖于 Python 著名的 **`qrcode`** 库，以及用于配合生成和处理图像的 **`Pillow`** (PIL) 库。
   - 调用原生的 `qrcode.QRCode()` 接口来轻松生成二维码图片数据，并可以通过图形界面展示及保存。
